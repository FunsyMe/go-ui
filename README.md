# go-ui
Simple UI panel for XRay and MTProto

> RU 🇷🇺

## Введение
### Для чего был создан go-ui?
- go-ui был создан, как независимая панель для поддержки многих протоколов VPN. Разработчики go-ui - сами владельцы VPN сервисов и знают, что удобнее пользователю.

### На чем это все разрабатывается?
- go-ui разрабатывается на GoLang. Для веб-интерфейса используется чистый HTML/CSS/JS

### Какие протоколы поддерживает go-ui?
- go-ui был создан относительно недавно - 18 июня, поэтому поддерживает только эти протоколы:
  - MTProxy (скоро)
  - SOCKS5 (скоро)
  - HTTP-прокси (скоро)
